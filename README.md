# pm2-events

catch pm2 processes events

## Usage

install

`npm install pm2-events --save`

example:

```
const pmEvents = require('./index');

let myPm = new pmEvents();

myPm
.on(function(e, p) {
    console.log(e); // event name
    console.log(p); // event detail
})
.error(function(err) {
    console.log(err);
});

```
